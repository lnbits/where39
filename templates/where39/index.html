Open Where39
Any3Words is good, but it's not private. Where39 is a BIP39 location protocol that you can run yourself.

Where39 is useful for dead drops, burying treasure, or just sharing your location with friends.

By running your own Where39 server, you can be sure that your location data is not shared with anyone.

1st word
First, we break the Earth into 2048 tiles using the BIP39 wordlist. This is achieved by dividing the Earth's latitude (180 degrees) and longitude (360 degrees) into a grid of 64x32 tiles, each covering 5.625 x 5.625 degrees.
2nd word
Each tile from the 1st level is further divided into 2025 smaller tiles (45x45), resulting in tiles that are 0.125 x 0.125 degrees each.
3rd word
These tiles are then split again into 2025 smaller tiles (45x45), creating tiles that are 0.00277777777 x 0.00277777777 degrees.
4th word
The next division produces another set of 2025 smaller tiles (45x45), with each tile measuring 0.00006172839 x 0.00006172839 degrees.
5th word
The 5th word, often called the "treasure word," is usually optional. However, if needed, the final division splits each tile into 400 smaller tiles (20x20), with each one being 0.00000308642 x 0.00000308642 degrees.
